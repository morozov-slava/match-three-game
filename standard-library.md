Класс `PlayingField` возможно немного переработать в части методов и повторно использовать в проектах, где есть потребность работы с двумерным массивом (т.к. в рамках проекта была использована эта структура данных для представления игрового поля).
*Переработать класс можно с использованием абстрагирования, выделив некоторый базовый класс для работы с двумерным массивом.*

Класс `ElementGenerator` можно в текущем виде повторно использовать в проектах, где необходимо с некоторой вероятностью генерировать некоторые произвольные объекты (значения).

Класс `Coordinates` можно также повторно использовать для задач, где объекты можно представлять на некоторой координатной оси.
*Переработать класс можно с использованием абстрагирования, выделив некоторый базовый класс для работы с N-мерным пространством.*

Класс `CombinationsCheckingPipeline` можно переработать и повторно использовать во множестве проектов, т.к. данный класс представляет собой реализацию паттерна "Pipeline".
*Переработать класс можно с использованием абстрагирования, выделив базовый класс который и будет представлять собой шаблон паттерна "Pipeline"*.
