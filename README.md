## Требования

- Python 3.10+
- pip



## Настройка и запуск проекта

1. Cоздайте виртуальное окружение

```bash
python -m venv .venv
```

2. Активируйте виртуальное окружение:

```bash
source .venv/bin/activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Запустите программу:

```bash
python main.py
```
