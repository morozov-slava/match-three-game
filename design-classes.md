`StatisticsDisplayer` - отображает игровую статистику (после хода или в конце игры) 
(можно реализовать через паттерн Observer)

`PlayerMoveMaker` - отвечает за выполнение игроком некоторого действия в рамках игрового процесса (сделать ход, запросить статистику, завершить игру, вернуть ход назад).
(можно реализовать через паттерн Command)

`PlayingFieldGenerator` - отвечает за генерацию игрового поля 
(возвращает объект класс `PlayingField`)
(можно реализовать через паттерн Factory)

`CombinationsChecker` - отвечает за поиск комбинаций на игровом поле.
Соответственно, принимает различные объекты, проверяющие игровое поле на предмет комбинаций.
(Можно реализовать через паттерн Command)
