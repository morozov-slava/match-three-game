Определим следующие кластеры на основе выделенных АТД (за исключением отсеянных):

- **playing_field.py**
	- `PlayingField` - класс, описывающий игровое поле:
		- выполняет операции вставки/удаления элементов по координатам поля;
		- выполняет операцию перестановки двух произвольных элементов на поле.
	- `ElementGenerator` - класс, генерирующий элементы игрового поля.
	- `PlayingFieldGenerator` - отвечает за генерацию игрового поля (возвращает объект класс `PlayingField`)
	  
- **statistics.py**
	- `StatisticsCounter` - класс для сбора игровой статистики во время игры. 
	- `StatisticsDisplayer` - отображает игровую статистику (после хода или в конце игры)

- **bonuses.py**
	- `Bonus` - класс, отвечающий за появление определённого бонуса.

- **game.py**
	- `GameStatus` - класс, проверяющий текущий статус игры
	- `PlayerMoveMaker` - отвечает за выполнение игроком некоторого действия в рамках игрового процесса (сделать ход, запросить статистику, завершить игру, вернуть ход назад).
