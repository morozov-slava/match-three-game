Перечень основных наиболее общих сущностей системы:

- **Игрок (Player)** - это сущность, непосредственно взаимодействующая с игрой.
  Игрок принимает решения по ходу игры (направляет запросы в систему) и получает от неё соответствующий отклик (сделать ход, выйти из игры, посмотреть статистику и т.п.)

- **Игровое поле (Playing Field)** - это сущность, описывающая основной компонент игры - матрицу, с которой взаимодействует игрок. В рамках этой сущностью происходит генерация игрового поля, изменение его состояния после хода игрока, а также его отображение
  
- **Элемент игрового поля (Playing Field Element)** - это сущность, описывающая отдельные компоненты игрового поля (их значение, местоположение и движение).
  
- **Игровая статистика (Game Statistics)** - это сущность, описывающая информацию, собираемую в процессе игры.
  
- **Система бонусов (Bonus System)** - это сущность, описывающая дополнительные возможности и бонусы, которые игрок может получить в процессе игры.
