Задание: Шаг 4 и Шаг 5

Кластер **playing_field.py**

Класс `PlayingField`:
- Конструктор:
	- Предусловие: размер игрового поля должен быть не меньше n (n = 1)
	- Постусловие: создан объект игрового поля заданного размера
- Команды:
	- `put`
		- Предусловие: ячейка пустая
		- Постусловие: в заданной ячейке появляется указанный элемент
	- `swap_two_elements`
		- Предусловие: обе ячейки не являются пустыми
		- Постусловие: элементы в двух ячейках меняются местами
	- `delete_element`
		- Предусловие: ячейка не является пустой
		- Постусловие: ячейка становится пустой
- Запросы:
	- `get_playing_field`
		- Постусловие: возвращает игровое поле
	- `get_shape`
		- Постусловие: возвращает размер игрового поля (длина и ширина)
	- `is_empty`
		- Постусловие: проверяет наличие элемента в указанной ячейке


Класс `ElementGenerator`:
- Конструктор:
	- Предусловие: размер набора возможных значений не должен быть меньше 1
	- Постусловие: создан объект, генерирующий значения из заданного набора
- Команды:
	- `generate`
		- Постусловие: генерация некоторого элемента из заданного набора по определённому правилу (логике)

Класс `PlayingFieldGenerator`
- Конструктор:
	- Предусловия: должны совпадать предусловия для объектов `PlayingField` и `ElementGenerator`.
	- Постусловие: создан объект генерирующий объект `PlayingField`, все поля которого заполнены по логике `ElementGenerator`.
- Команды:
	- `generate`:
		- Постусловие: создан объект `PlayingField` с заполненными ячейками.


Кластер **statistics.py**


Класс `StatisticsMetric`
- Конструктор:
	- Постусловие: создан объект для подсчёта определённой метрики игровой статистики
- Команды:
	- `increase`
		- Постусловие: значение метрики увеличено на указанное значение 
- Запросы:
	- `get_value`:
		- Постусловие: возвращает значение метрики


Класс `GameStatisticsCounter`
- Конструктор:
	- Постусловие: создан объект для хранения и подсчёта игровой статистики 
- Команды:
	- `add_metric`
		- Постусловие: добавлен объект класса `StatisticsMetric`
	- `increase`
		- Предусловие: данная метрика хранится в экземпляре
		- Постусловие: вызван метод `increase` у заданной метрики с указанными значением.
- Запросы:
	- `get_statictics`:
		- Постусловие: возвращает сводку по игровой статистике


Класс `GameStatisticsDisplayer`
- Конструктор:
	- Постусловие: создан объект для отображения игровой статистики в определённом формате
- Запросы:
	- `display`
		- Постусловие: отображает игровую статистику в заданном формате



Кластер **bonuses.py**

Класс `Bonus`
- Конструктор:
	- Постусловие: создан объект, описывающий некоторый игровой бонус
- Запросы:

		- Постусловие: возвращает параметры бонуса


Класс `BonusesChecker`
- Конструктор:
	- Постусловие: создан объект, проверяющий наличие определённых бонусов
- Команды:
	- `check`
		- Постусловие: проверяет наличие определённого бонуса в игре



Кластер **game.py**

Класс `InputCommand`
- Конструктор:
	- Постусловие: создан объект для исполнения хода игрока 
- Команды:
	- `run`
		- Постусловия: изменено состояние игрового поля


Класс `InputCommandsHandler`
- Конструктор:
	- Постусловие: создан объект для обработки команд игрока
- Команды:
	- `handle`
		- Постусловие: возвращает объект типа `InputCommand`


Класс `InputCommandsRunner`
- Конструктор:
	- Постусловие: создан объект для исполнения команд игрока
- Команды:
	- `run`
		- Предусловие: запрашиваемая команда предусмотрена правилами игры
		- Постусловие: выполняет вызов метода `run` у объекта типа `InputCommand`
